## Descripción del proyecto
Este proyecto es una práctica básica que combina Java y HTML.
Incluye un programa en Java que muestra un mensaje por consola y una página HTML sencilla que presenta información básica del equipo.

Su objetivo es practicar la estructura de un repositorio, el uso de GitHub y la creación de issues dentro de un proyecto en equipo.

## Ejecución archivo java
Abrimos en archivo en apache netbeans y lo ejecutamos haciendo click derecho sobre el proyecto y le damos a la opción de run file,
y se mostrara una pequeña consola de comandos donde se ejecutara el java.

## Ejecución archivo HTML
Ejecutamos el archivo html desde el navegador y a continuación de podra ver que pone titulo, el equipo y un parrafo.
